## Critical Bugs Found & Fixed
- **TG token conflict**: Claudette posting on HAL-O's bot — root cause was shared .env + session-lifecycle killing wrong processes
- **Auto-absorb killing wrong instance**: Detection matched ANY Claude process, not just same-instance. Fixed: match only `-n <instanceName>`
- **Self-kill x3**: Bulk process kill commands catching own PID. Hard rule added to CLAUDE.md
- **Response capture garbage**: Burrowing/Swooping/Fiddle-faddling from raw PTY. Fixed with @xterm/headless
- **Halo Chat chunk spam**: Every terminal buffer change became a chat message. Fixed: suppress chunks, send only final response
- **French voice**: Chatterbox is English-only. Fixed: skip for French, use Edge TTS Vivienne
- **Terminal scroll on Windows**: Wheel interceptor blocking mouse scroll. Fixed: macOS only

## Lessons Learned (HARD)
- **Never bulk kill processes** — always list PIDs, verify own PID, kill specific targets
- **Test before shipping** — 4 hours wasted on TG token issue because fixes weren't tested end-to-end
- **Two keys, two repos** — the TG isolation was fundamentally simple (Key A → Repo A, Key B → Repo B) but overcomplicated with auto-absorb/detection
- **Context degradation** — at 75%+ context, quality drops. Should flag and propose clean session earlier
- **Dev philosophy matters** — every shortcut was a rule violation
